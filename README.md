
# Cadastro de eventos

Projeto simples feito no curso "Introdução a JSF para aplicação webv" da Dio, usando JSF com PrimeFaces. Ao cadastrar eventos, os seus dados são exibidos na tabela presente na tela.


## Stack utilizada

**Front-end:** PrimeFaces, CSS

**Back-end:** Java, JSF
